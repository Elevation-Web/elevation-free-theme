A collection of powerful Gutenberg blocks designed to help nonprofit organizations build beautiful, accessible, and effective websites with ease.  

== Description ==  

Elevation Blocks provides a suite of custom Gutenberg blocks tailored for nonprofits, charities, and mission-driven organizations. Designed with accessibility, performance, and flexibility in mind, this plugin empowers users to create engaging content effortlessly.  

### **Features:**  
- Custom Gutenberg blocks optimized for nonprofit needs  
- Fully responsive and accessibility-friendly  
- Seamless integration with any WordPress theme  
- Lightweight and performance-focused  
- No coding required – easy drag-and-drop customization  

### **Why Use Elevation Blocks?**  
Nonprofits often struggle with complex website builders. Elevation Blocks simplifies content creation by offering pre-designed, flexible blocks that adapt to your cause’s needs. Whether you're showcasing success stories, calling for donations, or displaying upcoming events, this plugin makes it easier than ever.  

== Installation ==  

1. Upload the plugin files to the `/wp-content/plugins/elevation-blocks/` directory, or install the plugin directly from the WordPress plugin repository.  
2. Activate the plugin through the 'Plugins' screen in WordPress.  
3. Open the WordPress block editor and explore the new custom blocks under "Elevation Blocks."  
4. Drag, drop, and customize blocks to build your content.  

== Development Notes ==

All human-readable and uncompressed source files (JavaScript, SCSS, etc.) are located in the `/src` directory.

The plugin uses optimized and bundled files located in the `/build` directory for production.

Asset compilation and optimization are handled using [`@wordpress/scripts`](https://developer.wordpress.org/block-editor/reference-guides/packages/packages-scripts/). To rebuild assets during development, run: yarn or npm i. There are also dev and build scripts available on package.json file. 

Also there are in place composer libraries to check the code. Review composer.json file to check available scripts.

== Frequently Asked Questions ==  

= Does this plugin work with any theme? =  
Elevation Blocks is specifically designed to enhance the Elevation Free Theme by adding essential custom blocks.

= Will this slow down my site? =  
No. The plugin is built for performance, ensuring minimal impact on page speed.  

= Do I need coding knowledge to use this? =  
Not at all! Simply drag and drop the blocks in the Gutenberg editor and customize them to your needs.
