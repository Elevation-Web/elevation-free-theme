A modern, accessible, and performance-optimized WordPress theme designed for nonprofits, advocacy groups, and social impact organizations.  

== Description ==  

Elevation Free Theme is a beautifully designed, mission-driven WordPress theme built specifically for nonprofits, charities, and organizations striving to make a difference.  

With full **Gutenberg compatibility**, a clean and modern design, and a strong focus on **accessibility**, this theme ensures that your website looks great and is easy to navigate for all users.  

### Key Features:  
- **Responsive & Mobile-Friendly** – Looks great on all devices.  
- **SEO Optimized** – Get your cause noticed with built-in SEO best practices.  
- **Accessibility Ready** – Designed to meet WCAG standards for inclusivity.  
- **Fast & Lightweight** – Optimized for speed and performance.  
- **Translation & RTL Ready** – Reach a global audience with multilingual support.  

Whether you're running a small local charity or a global nonprofit, **Elevation Free Theme** gives you the tools to share your story and engage your supporters—without the technical hassle.  

Visit [https://github.com/Elevation-Web/elevation-free-theme/tree/main/wp-content/themes/elevation-free](https://github.com/Elevation-Web/elevation-free-theme/tree/main/wp-content/themes/elevation-free) for more details, documentation, and community support.

== Development Notes ==

All human-readable and uncompressed source files (JavaScript, SCSS, etc.) are located in the `/src` directory.

The plugin uses optimized and bundled files located in the `/build` directory for production.

Asset compilation and optimization are handled using [`@wordpress/scripts`](https://developer.wordpress.org/block-editor/reference-guides/packages/packages-scripts/). To rebuild assets during development, run: yarn or npm i. There are also dev and build scripts available on package.json file. 

Also there are in place composer libraries to check the code. Review composer.json file to check available scripts.
